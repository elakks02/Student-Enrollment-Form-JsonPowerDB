# Student Enrollment Form using JsonPowerDB

## Description

This project is a Student Enrollment Form developed using HTML, Bootstrap, JavaScript, jQuery, and JsonPowerDB.

The application stores student information in JsonPowerDB and supports:

- Save new student records
- Retrieve records using Roll Number
- Update existing student records
- Reset form to initial state
- Input validation

Database Used:

- Database Name: SCHOOL-DB
- Relation Name: STUDENT-TABLE
- Primary Key: Roll-No

---

## Benefits of using JsonPowerDB

1. Simple and lightweight database.
2. Schema-free JSON-based storage.
3. High performance and low latency.
4. Easy REST API integration.
5. Serverless architecture.
6. Reduces development time.
7. Supports real-time data operations.

## Project Workflow

### New Record

1. Enter Roll Number.
2. If Roll Number does not exist:
   - Enable data entry fields.
   - Enable Save button.
3. Fill all details.
4. Click Save.

### Existing Record

1. Enter Roll Number.
2. Existing data is automatically fetched.
3. Update required fields.
4. Click Update.

### Reset

1. Click Reset.
2. Form returns to initial state.
